# minilockcpp
multithread C++ minilock (cryptography) library port with a Qt5 GUI

## How to build

### Linux

```cmake
git clone https://github.com/mrom1/minilockcpp.git
cd minilockcpp
mkdir build && cd build
cmake ..
make
```
